Class: ILLMResponse

ILLMResponse(content, usage, rawResponseopt)

new ILLMResponse(content, usage, rawResponseopt)

Standardized response from an LLM provider. Encapsulates generated text along with usage metadata and original provider response.

Parameters:
Name Type Attributes Description
content string

Generated text from the model
usage object | null

Token usage statistics (provider-specific format)
rawResponse object <optional>

Original unmodified provider response

Example
const response = new ILLMResponse("Hello world", { total_tokens: 5 }, rawProviderData);
console.log(response.content); // "Hello world"
console.log(response.usage.total_tokens); // 5
